\section{Discussion}
\label{sec:disc}

% Key points:
%   1. Physical interpretation of the Affinity Tensor.
%   2. Relationship to existing entanglement measures (concurrence, entropy).
%   3. Limitations: RCT describes the relational structure; does not resolve
%      the measurement problem but provides a formal framework for it.
%   4. Future directions: experimental implementation, topological codes,
%      multi-qubit extension of U_RA.

The Affinity Tensor~$\alpha$ provides an operationalized measure of
relational coherence that is distinct from, though related to,
existing entanglement measures.
While concurrence~\cite{Wootters1998} and von Neumann entropy~\cite{vonNeumann1932}
quantify entanglement at a single moment, $\alpha(t)$ tracks the \emph{dynamics}
of relational coherence under environmental coupling~---~making it particularly
suited for characterizing gate performance in realistic noisy environments~\cite{Zurek2003}.

The \RCT\ does not resolve the quantum measurement problem; it provides
a formal relational framework within which the problem is precisely stated.
The philosophical implications of this framework, including the argument that
process ontology is formally required (not merely permitted) by quantum
mechanics, are developed in the companion paper~\cite{Adams2026Philosophy}.

Future directions include experimental implementation of~$\URA$ on
superconducting qubit platforms~\cite{Krantz2019}, extension to multi-qubit
relational networks, and investigation of topological surface code implementations
parameterized by the Affinity Tensor.
